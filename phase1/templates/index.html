AES-CBC Encryption — Step by Step

CBC = Cipher Block Chaining. Each block is chained to the previous one.
1 Split plaintext into 16-byte blocks
  "Hello, CBC World!!" → Block 0: "Hello, CBC World" | Block 1: "!!" + padding
2 Apply PKCS#7 padding to the last block
  If message = 11 bytes → add 5 bytes of value 0x05
  "Hello CBC" (9B) → "Hello CBC" + 07 07 07 07 07 07 07
3 For each block: XOR with IV (first) or previous ciphertext block
  XOR_input = PlaintextBlock ⊕ IV_or_PrevCiphertextBlock
4 AES-encrypt the XOR result to get the ciphertext block
  CiphertextBlock = AES_Encrypt(key, XOR_input)
5 That ciphertext block becomes "prev" for the next block

AES-CBC Decryption — Step by Step

1 AES-decrypt each ciphertext block (raw output = intermediate)
  AES_Raw = AES_Decrypt(key, CiphertextBlock)
2 XOR with IV (first block) or previous ciphertext block
  PlaintextBlock = AES_Raw ⊕ IV_or_PrevCiphertextBlock
3 On the last block: validate PKCS#7 padding
  Read last byte N. Check that the last N bytes all equal N.
  Valid:   ... 03 03 03  → strip 3 bytes → OK
  Invalid: ... 03 02 01  → PADDING ERROR → HTTP 403
4 The distinct HTTP response IS the oracle signal
HTTP 200 = padding valid    HTTP 403 = padding invalid
This single distinction allows full plaintext recovery without the key.
